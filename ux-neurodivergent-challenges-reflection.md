# Key UX Challenges & Considerations for Neurodivergent Users

## What common UX barriers affect neurodivergent users?
The most common UX barriers that affect neurodivergent users are cognitive overload, inconsistency, sensory overstimulation, and poor readability.

## How can predictability, clear feedback, and reduced distractions improve usability?
Predictability, clear feedback, and reduced distractions can improve usability by lowering the mental effort for users to complete tasks. This is especially
important for neurodivergent users, who are more likely to get distracted and lose interest.

## Why do overly complex interfaces cause issues for people with ADHD?
Overly complex interfaces cause people with ADHD to get easily distracted, overloaded, and lose interest.

## What are some real-world design examples that improve accessibility for neurodivergent users?
Some real-world design examples that improve accessibility for neurodivergent users are clear and predictable navigation, step-by-step guidance, and focus modes.

# Reflection

## How can Focus Bear’s design choices make tasks easier for neurodivergent users?
Focus Bear makes tasks easier for neurodivergent users by reducing distractions, providing structured routines, and breaking tasks into smaller steps.
Its predictable and guided design also supports users with ADHD or autism by making tasks feel clearer, calmer, and more manageable.

## If a feature is technically useful but cognitively overwhelming, how should it be redesigned?
If a feature is technically useful but cognitively overwhelming, it should be redesigned with a focus on lowering mental effort and improving clarity.
This can be done by breaking larger tasks into smaller ones, simplifying the interface, and reduce competing stimuli that can divide the user's attention.
By reducing cognitive load while preserving functionality, the feature becomes more accessible, usable, and less stressful especially for neurodivergent users.

## What are some small UX adjustments that can significantly improve user experience for neurodivergent users?
Some small UX adjustments that can significantly improve user experience, especially for neurodivergent users, is to have better default readability,
allow customization over sensory intensity, breaking chunky information, and improve visual hierarchy for easy scanning.
